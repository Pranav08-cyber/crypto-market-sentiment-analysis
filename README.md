---

## 📌 Project Overview

This project analyzes how **Bitcoin market sentiment** affects **trader performance** using two datasets:

### 📂 Datasets Used

| Dataset | Description |
|--------|------------|
| 📈 Hyperliquid Historical Data | Contains trader activity, positions, leverage, PnL, etc. |
| 📊 Fear & Greed Index | Daily Bitcoin sentiment classification (Fear / Greed) |

---

## 🎯 Objectives

- 💰 Analyze trader profitability under different sentiments  
- 📈 Study trading volume patterns  
- ⚠️ Evaluate risk-taking behavior  
- 🔄 Compare Buy vs Sell activity  
- 🧠 Understand trader psychology
